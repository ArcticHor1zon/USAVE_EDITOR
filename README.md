## How to build?

1. Clone the repository:

`git clone https://github.com/ArcticHor1zon/USAVE_EDITOR.git`\
`cd USAVE_EDITOR-master`

2. Build using cargo:

`cargo build --release`

3. Locate the binary:

The compiled binary is usually stored in `\target\release\`

## Usage

### Using a precompiled binary:
1. Go to Releases and download the latest precompiled build of the editor
2. Double-click the exe file
3. The editor should open
